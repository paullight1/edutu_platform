# Edutu Scholarship API — Docs (Mintlify)

This folder is the source for **docs.edutu.org**, built with
[Mintlify](https://mintlify.com). The full **API Reference** is generated
automatically from the backend's OpenAPI spec
(`GET https://api.edutu.org/v1/openapi.json`) — no hand-written endpoint pages.

## Structure

```
api-docs/
├── docs.json          # Mintlify config (nav, theme, OpenAPI source)
├── introduction.mdx
├── quickstart.mdx
├── authentication.mdx
├── rate-limits.mdx
├── errors.mdx
├── pagination.mdx
└── use-cases/
    ├── sync.mdx
    └── recommendations.mdx
```

The endpoint reference (`/api-reference/*`) is auto-generated by the `api`
block in `docs.json`, so it always matches the live API.

## Preview locally

```bash
npx mintlify@latest dev   # from this folder
```

## Deploy (docs.edutu.org)

1. Create a project at [Mintlify](https://dashboard.mintlify.com) and connect
   this GitHub repo, pointing the root at `edutu-platform/api-docs`.
2. Add the custom domain `docs.edutu.org` in the Mintlify dashboard.
3. Every push to `main` redeploys and re-fetches the OpenAPI spec.

## Keeping the reference in sync

The reference is pulled live from
`https://api.edutu.org/v1/openapi.json` at build time. If you prefer a
version-controlled snapshot (recommended for reproducible docs):

```bash
curl https://api.edutu.org/v1/openapi.json -o openapi.json
```

…then change `docs.json` → `"api": { "openapi": "openapi.json" }` and commit it.
