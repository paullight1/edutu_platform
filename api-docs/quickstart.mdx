# Quickstart

This guide walks through creating a key and calling the API.

## 1. Create a project and key

1. Sign in at [edutu.org](https://www.edutu.org/auth?mode=sign-up).
2. Open **Developer API** in the workspace menu.
3. Click **Create project**, choose an environment (`live` or `test`), and pick scopes.
4. Copy the key. **It is shown only once** and stored hashed on the server.

## 2. Make your first request

Pick the header style you prefer — all three are accepted.

<CodeGroup>
```bash cURL
curl https://api.edutu.org/v1/opportunities?limit=3 \
  -H "x-edutu-api-key: edu_live_xxxxxxxxxxxx"
```

```javascript JavaScript
const res = await fetch("https://api.edutu.org/v1/opportunities?limit=3", {
  headers: { "x-edutu-api-key": process.env.EDUTU_API_KEY },
});
const { data } = await res.json();
console.log(data);
```

```python Python
import os, httpx

r = httpx.get(
    "https://api.edutu.org/v1/opportunities",
    params={"limit": 3},
    headers={"x-edutu-api-key": os.environ["EDUTU_API_KEY"]},
)
print(r.json()["data"])
```
</CodeGroup>

## 3. Check your usage

```bash
curl https://api.edutu.org/v1/usage \
  -H "x-edutu-api-key: edu_live_xxxxxxxxxxxx"
```
